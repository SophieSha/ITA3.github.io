Scope and Limits

As for the limitations of our project we believe that actual creation of both the card and web application can be accomplished within our timeframe of 16 weeks. If not at worst we will extend it by a few months in case complications arise. The biggest factor however would be to have the cooperation with both the government and various companies to help build and support our project alongside its intended functions.

The limiting factor would be to gain cooperation with;

  • The government to ensure the project is approved and will be used throughout the various states. We believe this will be our hardest limit and challenge.
  • Various companies that ties with our hardware, and software functions such as: NXP (Hardware), Banks or Payment Methods (Direct Debit / BPAY / PayPal)

In terms of what can be achieved as mentioned we believe that both the card and application is able to be developed within the given 16 week time frame if no complications arise. This would include;

OMNI Card Development

  • Design: Logo, Colour Scheme of the Card
  • Program: Code we will be using and developing for the card to function with the readers
  • Hardware: MIFARE DESFire chip by NXP

OMNI Web Application Development

  • Design: Wireframes
  • Program: React’ Code to complement the application (HTML / JavaScript / CSS)

OMNI Functions

  • Transfer of funds from card to card
  • Group Functions – Management of multiple cards through one account
  • History Function
  • Multiple Payment Methods
