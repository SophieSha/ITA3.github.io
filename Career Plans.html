Career Plans

Daniel - Full Stack Developer

Becoming a full stack developer requires a decent working knowledge in a couple of programming languages. I plan on getting familiar with Client-side programming languages such as HTML, CSS and JavaScript through coursework and self study. This will give me the skills to products that are used by people in the real world i.e. A webpage or web application.

I would follow the same iterative process for developing my server-side programming language repertoire. I plan on learning to code in Java, Python and SQL. In doing so, I would be able to retrieve, modify and store data in databases at the behest of users.

Whilst learning languages is all well and good, they don’t prepare you for what it would be like to work in the industry and the tools that they use. To gain better insight into this, I plan on going to various industry talks held at Uni as well as gain part-time employment in the IT field. In doing so, I’d be exposed to real world practices and build a network.

By the time I graduate, I should have decent programming knowledge and some industry experience making it easier for me to gain full time employment in the field. By my estimation this plan should take about 3 - 5 years to complete.

Common Elements of Ideal Jobs

Because of ideal jobs requirements, most of group members are required to know basic IT knowledge and common front-end and back-end programming languages such as Java, HTML, CSS and Javascript, etc.

Both Davin and Daniel’s ideal jobs were to be an software developer or programmer. Daniel's ideal job is to be a Full Stack Developer, as a full stack software developer which use functional knowledge of the full stack to take a concept and transform it into a complete product. Meanwhile, Davin wishes to strive as a Java Developer. He would like to take part in the development of different types of programs or softwares ranging from a simple website to a program that could help develop prosthetic machinery or augmentation for humanity.

Wenshuo and Shane share similar interest in ideal jobs, which both related to web management and security. To become a system and networks administrator requires Wenshuo to work with complex network environment. It mainly focuses on management and maintenance of the network and system. Shane wants to be a cybersecurity and risk analyst. The daily aspects of the job would entail managing and monitoring the current security systems, detecting and fixing vulnerabilities, monitoring intrusion attempts, help and educate staff when they believe something is suspicious and when there is a suspected security breach.

All of our work positions requires teamwork, organizational skills, contributing and accomplishing different tasks with strong communication skills.

Differences of Ideal Jobs

In regarding to Daniel's ideal job, he has to see the whole picture of the program and be able to use multiple skills to construct and build the frame of the product at front-end and/or back-end platforms. But Davin's ideal job is a highly specialized position which is only focused on software development based on Java language. It requires large amount of deep study in Java area.

Wenshuo's working range as a network administrator is quite large and comprehensive but not very specific. The job is responsible for data update and security maintenance of the network site. It doesn’t really need high proficiency, but most of work must be properly understood. Shane's ideal job is more specialized than wenshuo's. Cybersecurity analyst must always be alert to security attacks from the network and ensure the advancement of security technology knowledge.

Sophie’s ideal job has the least similarity compared to others. Besides general understanding for software knowledge about UI/UX web and mobile App designing, it also requires Sophie to practice digital drawing skills and basic skill requirement of ‘Creativity’.
