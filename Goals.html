Goals

Creating a functional card

The first and most important goal will be to create a card capable of interfacing with existing readers found at stations, trams and buses.

Building the Web app

The web app will allow users to gain access to additional features not available to regular users. Some of the additional features will include;

  • Setting up automatic balance top up
  • Access to additional payment options
  • Transfer balance between cards owned by user
  • Manage family accounts

While the web application might not be absolutely necessary to begin using the solution, it provides tools that give users more control over their cards.
